# Miku Logic Paint S+ - Costume & Utility Patch

A customization tool for *Hatsune Miku Logic Paint S+*.

## Features
- **F2 Overlay**: Real-time costume customization menu.
- **Gender Overrides**: Mix and match Male/Female heads and bodies.
- **Persistence**: Settings are saved automatically.

## Quick Start
1. Download `winmm.dll` from the **Actions** tab.
2. Place `winmm.dll` into the game root.
3. Start the game.
   - *Note: The `mods/` folder and config will be generated automatically on first run.*

## Usage
- Press **F2** to open/close the overlay.
- Edit `mods/patch_config.ini` to toggle the hook debug console.

## Building
Requires MinGW-w64.
```bash
make
```
